# Wellspace API

PHP REST API for the [**Wellspace**](../../README.md) platform: pins, questionnaires, stations, translations, admin auth, and related services.

Deployed frontends (**feelvonRoll**, **wohlOpti**, …) consume the same codebase; tenants are differentiated by configuration and build-time branding under `brands/`.

See the repository root README for setup (`services/api`, Composer, schema, SMTP).
